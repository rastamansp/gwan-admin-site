gostei pode por favor adicionar a explicação deste fluxo no readme.md do projeto gwan-admin-site o qual descreva as etapas:


1.Cliente Cria login
1.1 cliente cria registro com nome, email e whatsapp
1.2 cliente recebe e-mail com codigo de ativação
1.3 cliente digita codigo e ativa sua conta e é direcionado para tela de login
1.4 cliente digita email na tela de login
1.5 cliente recebe email codigo de verificacao de login
1.6 cliente digita codigo de verificação de login e faz login na plataforma e é direcionado para ambiente logado tela dashboard.

2. Cliente cria Base de Conhecimento
2.1 cliente clica no botão de Adicionar Base
2.2 cliente preenche o nome e descrição da base de conhecimento
2.3 cliente clica no botão de Criar
2.4 cliente é redirecionado para a tela de upload de arquivo

3. Cliente faz upload de arquivo para base de conhecimento
3.1 cliente seleciona o arquivo pdf e clica no botão de Enviar
3.2 arquivo é adicionado a base de conhecimento

4. Cliente inicia processamento da base de conhecimento
4.1 cliente clica no botão de Iniciar Processamento
4.2 processo de extração de conhecimento é iniciado
